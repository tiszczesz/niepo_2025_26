Funkcje w JS

Definicja funkcji

Funkcja to blok kodu, który można wywołać w dowolnym miejscu w programie.

Tworzenie funkcji

Aby stworzyć funkcję, używamy słowa kluczowego function, a następnie podajemy nazwę funkcji oraz jej parametry w nawiasie.

function nazwaFunkcji(parametr1, parametr2) {
        // ciało funkcji
    }
